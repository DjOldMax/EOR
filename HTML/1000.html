6.1.1. Источники случайности в современных информационных системах

6.1.1. Источники случайности в современных информационных системах

Случайные числа находят самое широкое применение в современных информационных системах. Ниже приведены методы использования случайных чисел:

1) использование при реализации вероятностного шифрования;

2) использование для автоматической генерации сеансовых ключей шифрования;

3) генерация ключей и шифрование данных асимметричными алгоритмами;

4) использование для автоматической генерации ключей пользователей;

5) использование в качестве элементов криптографических протоколов, например протоколов аутентификации, протоколов обмена ключами, протоколов разделения секрета, протокола привязки к биту, протокола подбрасывания монеты, протоколов доказательств с нулевым разглашением, при выполнении ЭЦП и т. д.;

6) применение для генерации salt-последовательностей и векторов начальной инициализации для некоторых режимов применения шифров;

7) для применения в качестве составных элементов поточных шифров.

Все случайные числа, используемые для этих целей, разделяются на два больших класса − истинно случайные и псевдослучайные. Истинно случайные датчики случайных чисел используют некоторый физический источник энтропии для создания случайных чисел путем специальной обработки такого физического источника.

Такими источниками являются:

1) шумы младших разрядов некоторого АЦП (например оцифрованный сигнал микрофона звуковой карты);

2) тепловой шум в резисторе или диоде;

3) значение счетчика команд процессора;

4) детекторы событий ионизирующей радиации от некоторого источника;

5) аппаратные ГСЧ с встроенным физическим источником энтропии;

6) использование действий пользователя в качестве источника энтропии:

− координаты курсора мыши;

− величина паузы между нажатиями клавиш;

− взаимодействие между несколькими потоками программы;

7) вычисление хеш-функции от содержимого экрана;

8) вычисление хеш-функции от содержимого буфера подкачки.


Каждый из методов генерации имеет свои преимущества и недостатки.


Главным недостатком всех этих методов является медленная генерация энтропии, а значит и низкая скорость получения случайных чисел. В случаях, когда требуется получить большое количество случайных данных или в случае, когда последовательность случайных чисел должна иметь свойство воспроизводимости используют математические детерминированные алгоритмы получения псевдослучайных чисел (псевдослучайных последовательностей). Для инициализации ГПСЧ могут использоваться физические источники энтропии для получения последовательностей с лучшими характеристиками.

Свойство воспроизводимости может потребоваться, например, при построении потоковых шифров, работающих по принципу генератора гаммы.
