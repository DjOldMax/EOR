3.6.3. Использование методов линеаризации для решения систем нелинейных уравнений

Для систем нелинейных уравнений в общем виде не существует эффективных методов решения, но для разреженных сверхопределенных систем существует ряд так называемых методов линеаризации, т. е. преобразования нелинейных уравнений в линейные путем введения дополнительных переменных. Основываясь на последних достижениях в технике линеаризации, которая дает субэкспоненциальные алгоритмы для решения сверхопределенных (имеющих значительно больше уравнений чем неизвестных) систем квадратичных уравнений, N.Courtois и J.Pieprzyk доказали, что метод, называемый XSL, может обеспечить путь для эффективного решения уравнений этого типа и восстановить ключ из малого числа пар "открытый текст / шифртекст".

Преимуществом этого метода атаки, в отличие от стандартных статистических методов криптоанализа, являются следующие особенности:

− он требует достаточно малого количества пар "открытый текст / шифртекст";

− его сложность растет субэкспоненциально с ростом числа раундов.

Несмотря не перспективы алгебраического анализа, до сих пор не было продемонстрировано атак этого типа на полные версии блочных шифров. Их исследование находится все еще на ранней стадии, точная сложность этого метода пока полностью не определена и остаются открытыми много вопросов связанных с его применимостью.

Существует две разновидности алгебраической атаки. Первая − не использует ключевое расписание и рассматривает раундовые подключи как полностью независимые случайные последовательности. Вторая − включает в общую систему уравнений уравнения для ключевого расписания и имеет меньшую сложность решения. Первая из них требует количество пар "открытый текст/шифртекст", равное числу раундов или более для проведения атаки, для второй достаточно одной-двух в качестве исходных данных. В качестве примера второго сценария задача восстановления ключа 128-битного AES может быть записана как система из 8000 квадратичных уравнений с 1600 переменными.

Алгоритм XL состоит из следующих шагов:

1. Построение исходной системы квадратичных уравнений.

2. Для каждой переменной, входящей в уравнения исходной системы, отбираются уравнения, которые содержат члены 2-й степени только с участием выбранной переменной, и при этом содержат любые члены первой степени, а также, возможно единичную константу.

3. Умножаем отобранные уравнения для каждой переменной на эту переменную, в результате чего получаем в ряде случаев набор новых линейно-независимых уравнений.

4. Для вновь полученных уравнений определяем одночлены, которые не участвуют в их записи, и подставляем вместо одночленов нового уравнения выражения, полученные из исходного набора уравнений для данной переменной. В результате этого в ряде случаев получаются дополнительные линейно-независимые уравнения.

5. Повторяем шаги 2−4 до получения системы с рангом, равным количеству различных одночленов, присутствующих в исходной системе или до тех пор, пока не останется ни одного уравнения, удовлетворяющего условию п. 2.

6. Выполняем замену одночленов 2-й степени путем введения новых переменных, после чего решаем систему линейных уравнений методом Гаусса.

Кроме алгоритма XL был предложен другой алгоритм XSL, что означает «extended sparse linearization». В алгоритме XL требуется умножить каждое уравнение на все возможные одночлены некоторой степени. Вместо этого оказывается выгодным использовать не все переменные, а умножать их на тщательно выбранные одночлены. Наилучшие результаты приносит использование произведения одночленов, которые уже имеются в других уравнениях.

Конечной целью этих алгоритмов является примерное соответствие количества уравнений в системе количеству используемых в них одночленов. Если число уравнений немного меньше числа одночленов, то система имеет множество решений, которые в некоторых случаях могут быть найдены с помощью полного перебора значений недостающих переменных.
