6.2.3. Атаки на стегосистемы

6.2.3. Атаки на стегосистемы

Специалистов в области стегоанализа называют стегоаналитиками и их основной задачей является вскрытие стегосистемы с помощью различных методов атаки.

К основным угрозам безопасности для стегосистем относятся:

• обнаружение стеганографического канала − защита от этой угрозы является основной задачей стеганографии;

• извлечение скрытого сообщения − противник не просто обнаружвает сообщение, но может отделить данные стеганографического контейнера от других данных;

• разрушение скрытого сообщения − внесение таких изменений, не нарушающих естественности контейнера, при которых извлечь данные из контейнера становится невозможно;

• подмена скрытого сообщения другим − наиболее сильная угроза стеганографической системе.


Атаки, направленные на обнаружение

• Атака с известным контейнером − у атакующего имеется исходный файл контейнера и заполненный информацией контейнер, стегоканал обнаруживается путем сравнительного анализа двух контейнеров.

• Атака с выбором контейнера − злоумышленник выбирает контейнер и вынуждает воспользоваться именно этим контейнером для передачи сообщения.

• Атаки с известными данными до упаковки в контейнер − атакующему известны данные для передачи, но неизвестно, как они упакованы в контейнер, и есть ли они там вообще.

• Атаки с выбранными данными до упаковки в контейнер.

• Атака с известным скрытым сообщением − может быть как с известными исходными данными (до упаковки), так и без них. В первом случае атака состоит в попытке получения секретного ключа.

• Атака с выбором скрытого сообщения − наиболее благоприятный вариант для аналитика.

• Атака на основе известной математической модели контейнера.


Атаки, направленные на уничтожение

• Сжатие с потерей данных − для уменьшения различных мультимедийных файлов часто используется сжатие с потерей данных − это jpeg для картинок, mp3 для звука, mpeg для видеофайлов и множество других. Для многих методов встраивания сообщений такое сжатие приводит к искажению и даже полной потере сообщения.

• Геометрические преобразования графических данных:

− уменьшение размера контейнера путем обрезания граничных областей;

− поворот изображения на малый угол, или несколько взаимнокомпенсирующихся поворотов;

− масштабирование с различными коэффициентами.

• Внесение в контейнер дополнительного случайного шума.

• Фильтрация мультимедийных данных.

• Изменение гистограммы изображения.

• Комбинированные атаки.
